Apa itu Array?

Array adalah struktur data yang dapat menyimpan sekumpulan elemen dengan tipe data yang sama dalam satu nama variabel.

  • âœ… Semua elemen bertipe data sama

Fungsi yang Mengembalikan Nilai

int hitungJumlah(int arr[], int n) {
    int total = 0;
    for (int i = 0; i < n; i++) {
        total += arr[i];
    }
    return total;
}

double hitungRataRata(int arr[], int n) {
    int total = hitungJumlah(arr, n);
    return (double)total / n;
}

Mencari Maksimum dan Minimum

int cariMaksimum(int arr[], int n) {
    int maks = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > maks) {
            maks = arr[i];
        }
    }
    return maks;
}

int cariMinimum(int arr[], int n) {
    int min = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] < min) {
            min = arr[i];
        }
    }
    return min;
}

Membalik Urutan Array

void balikArray(int arr[], int n) {
    int awal = 0;
    int akhir = n - 1;
    
    while (awal < akhir) {
        // Tukar elemen
        int temp = arr[awal];
        arr[awal] = arr[akhir];
        arr[akhir] = temp;
        
        awal++;
        akhir--;
    }
}

Pencarian Berurutan: Memeriksa setiap elemen satu per satu dari awal hingga akhir

  • âœ… Sangat sederhana
  • âœ… Bekerja pada array tidak terurut
  • âš ï¸ Kompleksitas: O(n)

Visualisasi Sequential Search

Implementasi Sequential Search

int sequentialSearch(int arr[], int n, int kunci) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == kunci) {
            return i;  // Ditemukan, return indeks
        }
    }
    return -1;  // Tidak ditemukan
}

int main() {
    int data[] = {10, 25, 33, 40, 55, 67, 78};
    int hasil = sequentialSearch(data, 7, 40);
    
    if (hasil != -1)
        cout << "Ditemukan pada indeks " << hasil;
    else
        cout << "Tidak ditemukan";
}
