## B. CAPAIAN PEMBELAJARAN MATA KULIAH (CPMK)

**CPMK 1:** Mahasiswa mampu menjelaskan konsep dasar pemrograman dan metodologi penyelesaian masalah menggunakan bahasa pemrograman C++.

**CPMK 2:** Mahasiswa mampu menerapkan struktur pemrograman dasar (variabel, tipe data, operator, ekspresi, input/output, percabangan, perulangan) dalam bahasa C++.

**CPMK 3:** Mahasiswa mampu mengimplementasikan konsep fungsi, array, pointer, dan tipe data bentukan (struct) untuk menyelesaikan masalah komputasi.

**CPMK 4:** Mahasiswa mampu menerapkan teknik debugging, error handling, dan exception handling dalam pengembangan program C++.

**CPMK 5:** Mahasiswa mampu mengimplementasikan file handling dan memahami konsep dasar pemrograman berorientasi objek menggunakan C++.

---

## C. SUB-CAPAIAN PEMBELAJARAN MATA KULIAH (SUB-CPMK)

**Sub-CPMK 1.1:** Mahasiswa mampu menjelaskan konsep dasar pemrograman, paradigma pemrograman, dan proses penyelesaian masalah secara komputasional.

**Sub-CPMK 1.2:** Mahasiswa mampu mengidentifikasi langkah-langkah dalam proses pemecahan masalah dengan pendekatan algoritmik.

**Sub-CPMK 1.3:** Mahasiswa mampu memahami penerapan C++ di berbagai platform dan domain aplikasi.

**Sub-CPMK 2.1:** Mahasiswa mampu menerapkan tipe data, variabel, konstanta, dan operator dalam C++.

**Sub-CPMK 2.2:** Mahasiswa mampu mengimplementasikan operasi input/output dalam program C++.

**Sub-CPMK 2.3:** Mahasiswa mampu mengembangkan program dengan struktur percabangan (if, if-else, switch).

**Sub-CPMK 2.4:** Mahasiswa mampu mengembangkan program dengan struktur perulangan (for, while, do-while).

**Sub-CPMK 3.1:** Mahasiswa mampu menerapkan fungsi (termasuk parameter passing, return value, dan fungsi rekursi) dalam program C++.

**Sub-CPMK 3.2:** Mahasiswa mampu mengimplementasikan array (1D dan multidimensi) dan operasinya dalam program C++.

**Sub-CPMK 3.3:** Mahasiswa mampu menerapkan konsep pointer dan penggunaannya dalam program C++.

**Sub-CPMK 3.4:** Mahasiswa mampu mengimplementasikan struct dan tipe data bentukan dalam program C++.

**Sub-CPMK 4.1:** Mahasiswa mampu mengidentifikasi dan memperbaiki error dalam program menggunakan teknik debugging.

**Sub-CPMK 4.2:** Mahasiswa mampu menerapkan exception handling untuk penanganan error dan exceptional conditions dalam program C++.

**Sub-CPMK 5.1:** Mahasiswa mampu mengimplementasikan file handling untuk membaca dan menulis data dalam program C++.

**Sub-CPMK 5.2:** Mahasiswa mampu memahami konsep dasar pemrograman berorientasi objek (class, object, inheritance, polymorphism).
